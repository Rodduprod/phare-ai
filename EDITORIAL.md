## 🎯 Guidelines rédactionnelles

### Style Le Labo AI
- **Accessible** : pas de jargon technique inutile
- **Pratique** : exemples concrets, applications réelles
- **Enthousiasmant** : l'IA c'est passionnant !
- **Honnête** : limites et risques aussi

### Structure type
1. **Contexte** (1-2 paragraphes)
2. **Nouveautés/Points clés** (sections détaillées)
3. **Applications pratiques** 
4. **Ce que ça change** (impact, conclusion)

### SEO
- **Titre** : clair, avec mot-clé principal
- **Description** : 150-160 caractères, engaging
- **Tags** : 2-4 tags pertinents (modèles, outils, agents, etc.)
- **Image** : couverture 1200x630px
- **Mots-clés** : naturels, pas de keyword stuffing

### Longueur cible
- **Articles deep** : 1200-1800 mots  
- **Articles quick** : 600-900 mots
- **Tutorials** : 1000-1500 mots
