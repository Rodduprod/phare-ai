## Votre première automatisation en 15 minutes

Exemple concret : recevoir un email avec un formulaire de contact → ajouter automatiquement le contact dans une feuille Google Sheets.

**Sur Zapier :**
1. Créez un compte gratuit sur zapier.com
2. Cliquez "Create Zap"
3. Choisissez le déclencheur : "Gmail" → "New Email"
4. Configurez le filtre : emails avec "Formulaire de contact" dans le sujet
5. Choisissez l'action : "Google Sheets" → "Create Spreadsheet Row"
6. Mappez les champs : nom → colonne A, email → colonne B, message → colonne C
7. Testez avec un vrai email
8. Activez le Zap

C'est tout. Le Zap tourne en arrière-plan 24h/24.

## Ajouter l'IA dans le workflow

Le même workflow, mais maintenant l'IA analyse le message avant de l'ajouter :

**Ajoutez une étape entre l'email et Sheets :**
- Action : "ChatGPT" (dans Zapier)
- Prompt : "Analyse ce message de contact et détermine : 1) le type de demande (commercial, support, autre), 2) l'urgence (haute/normale/basse). Réponds en JSON."
- Puis mappez aussi ces champs dans Sheets

Résultat : chaque nouveau contact est automatiquement classifié. Vous voyez en un coup d'œil ce qui est urgent.

## Make vs Zapier — quand choisir ?

**Zapier** si vous débutez, si vous avez besoin de connecter des outils classiques rapidement, si vous êtes prêt à payer un peu plus pour la simplicité.

**Make** si vous voulez créer des workflows complexes avec des boucles et des conditions, si le volume est important (Make facture par opération, moins cher), si vous aimez visualiser l'ensemble du flux.

## Ce qu'il faut retenir

Créez votre premier Zap sur un problème concret et simple. Testez avant d'activer. Ajoutez l'IA comme étape intermédiaire pour enrichir les données. Déboguez toujours en consultant les logs — pas en devinant.
