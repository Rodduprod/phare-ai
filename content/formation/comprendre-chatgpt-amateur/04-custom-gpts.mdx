## Qu'est-ce qu'un Custom GPT ?

OpenAI a ouvert la possibilité de créer des versions personnalisées de ChatGPT, appelées **Custom GPTs** (ou "Mes GPTs"). En clair : vous configurez un assistant avec des instructions précises, des connaissances spécifiques, et des capacités particulières — et vous l'utilisez (ou le partagez) comme si c'était une app.

Pas de code. Une interface de configuration en langage naturel. Disponible pour tous les abonnés ChatGPT Plus.

---

## Ce que vous pouvez personnaliser

### Instructions permanentes (System prompt)
C'est le comportement de base du GPT. Tout ce que vous mettez ici est présent dans chaque conversation.

Exemples :
- "Tu es un assistant spécialisé dans le droit immobilier français. Tu réponds uniquement aux questions juridiques liées à l'immobilier. Pour toute question hors sujet, tu renvoies à un avocat."
- "Tu parles toujours en tutoiement. Tu es direct et concis. Tu utilises des exemples concrets. Tu ne dis jamais 'bien sûr'."

### Connaissances (Knowledge files)
Vous pouvez uploader des fichiers (PDF, Word, texte) que votre GPT utilisera pour répondre. Par exemple :
- Votre charte éditoriale
- Votre catalogue produit
- Les FAQ de votre service client
- Votre guide de style

Le GPT s'appuie sur ces documents pour répondre plutôt que sur ses connaissances générales — ce qui réduit les hallucinations sur vos données spécifiques.

### Capacités
Vous pouvez activer ou désactiver :
- **Navigation web** — le GPT peut chercher des infos récentes
- **Génération d'images** (DALL-E) — il peut créer des visuels
- **Exécution de code** — il peut faire des calculs, analyser des données

### Actions (avancé)
Vous pouvez connecter votre GPT à des APIs externes pour qu'il accède à vos propres données en temps réel. Cette partie nécessite des notions techniques.

## Créer votre premier Custom GPT

1. Allez sur chatgpt.com → cliquez sur votre profil → "Mes GPTs" → "Créer un GPT"
2. L'assistant de création vous pose des questions : "À quoi sert ton GPT ?"
3. Répondez en langage naturel — l'assistant configure automatiquement les instructions de base
4. Affinez dans l'onglet "Configurer" si nécessaire
5. Publiez (pour vous seul, ou partagez avec un lien, ou publiez publiquement)

## Exemples de Custom GPTs utiles

**Assistant éditorial personnel**
Instructions : ton de rédaction précis, mots interdits (ceux que vous n'utilisez jamais), public cible de vos contenus, structure de vos articles. Knowledge : vos meilleurs articles passés.
→ Génère du contenu dans *votre* style, pas dans le style générique de GPT

**Expert FAQ client**
Knowledge : toutes vos FAQ, conditions générales, politique de retour, fiches produit.
→ Répond aux questions clients en s'appuyant uniquement sur vos docs

**Préparateur de réunion**
Instructions : à partir d'un ordre du jour, génère des questions pertinentes, anticipe les objections, propose un plan de présentation.
→ Utilisez-le avant chaque réunion importante

**Coach CV et lettre de motivation**
Instructions : expert RH, focus sur le marché français, conseil personnalisé et direct. 
→ Un outil que vous pouvez partager avec votre équipe ou des candidats

## Les limites

- Les fichiers knowledge ont une taille limite (~500 pages)
- Le GPT ne peut pas accéder à des données en temps réel sans configuration d'actions (avancé)
- Les abonnés free n'ont accès qu'à des Custom GPTs partagés, pas à la création
- Si vous partagez votre GPT publiquement, OpenAI peut voir vos instructions — ne mettez pas d'informations confidentielles dedans

---

## Ce qu'il faut retenir

- Un Custom GPT est une version spécialisée de ChatGPT que vous configurez sans coder
- Les instructions permanentes définissent le comportement, les fichiers knowledge donnent des connaissances spécifiques
- Vous pouvez le garder privé, le partager par lien, ou le publier publiquement
- Ne mettez pas d'informations confidentielles dans les instructions d'un GPT public

Dernière leçon de ce module : les limites et biais de ChatGPT — ce que vous devez savoir pour l'utiliser avec discernement.
