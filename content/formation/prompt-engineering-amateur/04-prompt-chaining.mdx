## Pourquoi un seul prompt ne suffit pas toujours

Imaginons : vous voulez analyser 10 retours clients, extraire les thèmes principaux, identifier les problèmes critiques, puis rédiger une synthèse pour votre équipe produit.

Tout mettre dans un seul prompt ? Possible. Mais vous perdez le contrôle à chaque étape. Si l'analyse des thèmes est mauvaise, la synthèse finale l'est aussi — sans savoir où ça a dérapé.

Le prompt chaining découpe le problème en étapes vérifiables.

## Exemple : de la recherche à l'article

**Étape 1 — Extraction**
> "Voici 5 articles sur le sujet X. Extrais les 10 points clés les plus importants. Format : liste numérotée."

*Vous vérifiez que les 10 points sont bons.*

**Étape 2 — Structure**
> "Voici 10 points clés sur X. Crée un plan d'article en 4 parties qui les organise logiquement."

*Vous validez le plan.*

**Étape 3 — Rédaction**
> "Voici le plan. Rédige la partie 1 en 400 mots, ton direct, exemples concrets."

Chaque étape est contrôlable. Si l'étape 2 est mauvaise, vous ne perdez pas l'étape 3.

## Le prompt système — persistant dans la session

Avec l'API (ou dans les instructions personnalisées de ChatGPT), le prompt système définit le comportement de l'IA pour toute la session :

```
Tu es un assistant éditorial pour un média tech francophone.
Ton : direct, un peu sceptique, exemples concrets.
Tu n'utilises jamais : "il est important de noter", "n'hésitez pas à", "dans un monde en constante évolution".
Tu réponds en français.
```

Vous n'avez plus à répéter ces instructions à chaque prompt.

## Automatiser le chaining

Avec des outils comme Make ou Zapier, vous pouvez enchaîner automatiquement des prompts :
- Nouveau email → extraire le type de demande → classifier → générer une ébauche de réponse
- Nouvel article RSS → résumer → traduire → poster sur Slack

Le chaining n'est pas réservé aux sessions de chat. C'est la base de l'automatisation IA sérieuse.

## Ce qu'il faut retenir

Tâche complexe = plusieurs prompts séquentiels. Valider chaque étape avant de passer à la suivante. Le prompt système évite les répétitions dans une session longue. Le chaining automatisé, c'est le début de l'automatisation IA réelle.
